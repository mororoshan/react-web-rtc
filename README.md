# react-web-rtc

React + TypeScript application exploring **WebRTC**: peer connections, media streams, and signaling for real-time communication.

Part of a larger WebRTC stack — see also [pwa-webrtc-nodejs-react](https://github.com/mororoshan/pwa-webrtc-nodejs-react) and [pwa-nodejs-signal-server](https://github.com/mororoshan/pwa-nodejs-signal-server).

## Stack

- React, TypeScript
- WebRTC API (RTCPeerConnection, media tracks)
- Signaling over WebSocket

## Getting started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000).

For full P2P flow, run the [signaling server](https://github.com/mororoshan/pwa-nodejs-signal-server) locally.
